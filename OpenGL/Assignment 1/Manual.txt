[OpenGL Application]

This application displays four shapes simultaneously and handles their motions

* Four shapes lie on four quarters, and to select a shape(or a quarter), you can use either of the following:
- Left click on a shape(a quarter)
- Right click anywhere in the window and choose a quarter from the menu
- Using F1 - F4 , mapped as following
			-----------------
			|		|		|
			|  F2	|   F1	|
			|		|		|
			-----------------
			|		|		|
			|  F3	|   F4	|
			|		|		|
			-----------------

* use [Up, Down, Left, Right] keys, you can rotate selected shape (Up/Down for X-axis | Left/Right for Y-axis)
	|- use HOME key to activate rotation round Z-axis where Up/Left same direction and so Down/Right

* use F5 on a selected shape to make it rotate automatically
	|- pressing F5 makes it available to use controlling keys (mentioned above) to control rotation direction
